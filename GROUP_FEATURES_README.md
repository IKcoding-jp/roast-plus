# グループ機能 使い方ガイド

## 概要

BYSN業務アプリに追加されたグループ機能により、複数の端末でデータを共有・同期できるようになりました。Googleアカウントを使用して、ゲームのクランのようなグループシステムを実現しています。

## 主な機能

### 1. グループ管理
- **グループ作成**: リーダーとして新しいグループを作成
- **メンバー招待**: メールアドレスでメンバーを招待
- **権限管理**: リーダーとメンバーの役割分け
- **グループ編集**: グループ名や説明の変更

### 2. 権限システム
- **リーダー**: グループの編集、メンバーの招待・削除、権限変更、設定管理が可能
- **メンバー**: 設定された権限に基づいてデータの閲覧・編集が可能
- **詳細権限設定**: リーダーが各データタイプとメンバー権限を細かく設定可能

### 3. データ同期
- **焙煎記録**: グループ内で焙煎記録を共有
- **TODOリスト**: スケジュールやタスクの共有
- **ドリップカウンター**: ドリップパックの記録を共有
- **担当表**: メンバーとラベルの設定を共有
- **スケジュール**: 焙煎スケジュールを共有

## 使い方

### グループ管理にアクセス
1. アプリを起動
2. 左上のハンバーガーメニューをタップ
3. 「グループ管理」を選択

### グループを作成
1. グループ管理画面で「+」ボタンをタップ
2. グループ名と説明を入力
3. 「グループを作成」をタップ
4. 作成者は自動的にリーダーになります

### メンバーを招待
1. グループ詳細画面で「招待」ボタンをタップ
2. 招待したいユーザーのメールアドレスを入力
3. 「招待を送信」をタップ
4. 招待されたユーザーは7日間以内に参加を選択できます

### 招待に参加
1. グループ管理画面で招待通知を確認
2. 「確認」ボタンをタップ
3. 招待一覧で「参加」ボタンをタップ
4. グループにメンバーとして参加

### データを同期
1. グループ詳細画面で「データ同期」セクションを確認
2. **データを同期**: 自分のデータをアップロードし、グループのデータをダウンロード

### メンバー管理（リーダーのみ）
1. グループ詳細画面でメンバー一覧を確認
2. メンバーの「...」ボタンをタップ
3. 以下の操作が可能：
   - **リーダーに昇格**: メンバーをリーダーに変更
   - **メンバーに降格**: リーダーをメンバーに変更
   - **削除**: グループからメンバーを削除

### グループ設定（リーダーのみ）
1. グループ詳細画面で「グループ設定」をタップ
2. **データ権限設定**: 各データタイプの編集権限を設定
   - リーダーのみ: リーダーだけが編集可能
   - 全メンバー: 全メンバーが編集可能
   - 閲覧のみ: 誰も編集できない
3. **メンバー権限設定**: メンバーの各種権限を設定
   - メンバー招待権限: メンバーが他のユーザーを招待できるか
   - データ同期権限: メンバーがデータ同期を実行できるか
   - メンバー一覧閲覧権限: メンバーがメンバー一覧を見れるか
4. 「保存」ボタンで設定を保存

### グループから脱退
1. グループ詳細画面の下部にある「グループから脱退」ボタンをタップ
2. 確認ダイアログで「脱退」を選択
3. **注意**: リーダーは他のリーダーに権限を譲ってから脱退する必要があります

## データ同期の詳細

### 同期されるデータ
- **焙煎記録**: 日時、豆の種類、焙煎度合い、メモなど
- **TODOリスト**: 本日のタスクとスケジュール
- **ドリップカウンター**: ドリップパックの使用記録
- **担当表**: メンバーとラベルの設定
- **スケジュール**: 焙煎スケジュール

### 同期の仕組み
- **同期**: 自分のデータをグループにアップロードし、グループのデータをダウンロード
- **双方向同期**: 一つのボタンで完全なデータ同期を実行
- **権限ベース同期**: 設定された権限に基づいてデータの編集・同期を制御
- **リアルタイム監視**: グループデータの変更をリアルタイムで監視（今後の機能）

## セキュリティ

### アクセス制御
- Googleアカウントによる認証
- グループメンバーのみがデータにアクセス可能
- 招待は7日間で期限切れ

### データ保護
- Firestoreのセキュリティルールによる保護
- ユーザーごとのデータ分離
- グループ内での権限管理

## トラブルシューティング

### よくある問題

**Q: 招待が届かない**
A: メールアドレスが正しく入力されているか確認してください。招待は7日間で期限切れになります。

**Q: データが同期されない**
A: インターネット接続を確認してください。また、グループメンバーであることを確認してください。

**Q: リーダー権限がない**
A: リーダーのみがグループの編集やメンバー管理ができます。現在のリーダーに権限の変更を依頼してください。

**Q: グループから脱退できない**
A: リーダーの場合、他のリーダーに権限を譲ってから脱退する必要があります。
