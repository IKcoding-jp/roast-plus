# Google Search Console 設定ガイド

## 1. Google Search Consoleにアクセス
https://search.google.com/search-console にアクセスし、Googleアカウントでログインします。

## 2. プロパティを追加
1. 「プロパティを追加」をクリック
2. 「URLプレフィックス」を選択
3. 以下のURLを入力：
   ```
   https://bysnlogapp.web.app
   ```

## 3. 所有権の確認
以下のいずれかの方法で所有権を確認：
- **HTMLファイル**: `google1234567890.html` ファイルをダウンロードし、`web/` フォルダに配置して再デプロイ
- **HTMLタグ**: 提供されたmetaタグを`web/index.html`の`<head>`セクションに追加

## 4. Sitemapの送信
1. 左メニューから「Sitemap」を選択
2. 「新しいSitemapを追加」をクリック
3. 以下のURLを入力：
   ```
   https://bysnlogapp.web.app/sitemap.xml
   ```

## 5. インデックス登録の促進
1. 「URL検査」ツールを使用
2. メインURL（https://bysnlogapp.web.app）を入力
3. 「インデックス登録をリクエスト」をクリック

## 6. 追加のSEO最適化

### パフォーマンス改善
- Core Web Vitalsの監視
- モバイルユーザビリティの確認

### コンテンツ最適化
- 関連キーワードの特定

## 7. 定期的な監視
