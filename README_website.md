# Roast Plus Webサイト

Roast Plusアプリの公式Webサイトです。BYSNで働く人のための業務サポートアプリとして、プライバシーポリシー、利用規約、アプリ紹介ページを含んでいます。

## 📁 ファイル構成

```
├── index.html              # メインランディングページ
├── privacy_policy.html     # プライバシーポリシー
├── terms_of_service.html   # 利用規約
└── README_website.md       # このファイル
```

## 🌐 ページ概要

### 1. メインページ (index.html)
- アプリの紹介と主要機能の説明
- 美しいアニメーションとレスポンシブデザイン
- プライバシーポリシーと利用規約へのリンク

### 2. プライバシーポリシー (privacy_policy.html)
- 個人情報の収集・利用について
- データの保存期間とセキュリティ対策
- BYSN社員向けの利用対象
- お客様の権利とお問い合わせ方法

### 3. 利用規約 (terms_of_service.html)
- アプリの利用条件
- 禁止事項と利用制限
- 料金体系と支払い方法
- 免責事項と準拠法

## 🎨 デザイン特徴

- **モダンなグラデーション**: 美しい紫系のグラデーションを使用
- **レスポンシブデザイン**: スマートフォン、タブレット、PCに対応
- **アニメーション**: スクロール時のフェードイン効果
- **アクセシビリティ**: 読みやすいフォントとコントラスト
- **日本語最適化**: 日本語フォントとレイアウトに最適化

## 🚀 使用方法

### ローカルでの確認
1. ファイルをWebサーバーにアップロード
2. ブラウザで `index.html` にアクセス
3. 各ページ間のリンクが正常に動作することを確認

### オンライン公開
1. GitHub Pages、Netlify、Vercel等のホスティングサービスにアップロード
2. カスタムドメインの設定（オプション）
3. SSL証明書の設定（推奨）

## 🔧 カスタマイズ

### 色の変更
CSSファイル内の以下の変数を変更することで、色をカスタマイズできます：

```css
/* メインカラー */
--primary-color: #667eea;
--secondary-color: #764ba2;

/* 背景グラデーション */
background: linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%);
```

### コンテンツの更新
各HTMLファイル内のコンテンツを編集することで、情報を更新できます。
